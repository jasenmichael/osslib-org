
# Open Source Software Library - front-end

live site here [osslib.org](http://osslib.org)

![dropstagrambox-screenshot](http://jasenmichael.com/img/osslib-org.jpg)

#### to install locally

```
git clone https://github.com/jasenmichael/osslib-org.git

cd osslib-org

npm install

npm run dev
```
